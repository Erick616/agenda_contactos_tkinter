# Agenda de contactos con Tkinter

Aplicación de escritorio desarrollada en Python con Tkinter para administrar contactos mediante una interfaz gráfica.

## Funciones

- Registrar contactos
- Editar contactos
- Eliminar contactos
- Buscar por nombre, teléfono o correo
- Mostrar todos los contactos
- Validar teléfono y correo
- Evitar contactos duplicados
- Editar con doble clic
- Guardar la información en JSON
- Recuperar los contactos al abrir nuevamente la aplicación

## Tecnologías utilizadas

- Python
- Tkinter
- JSON
- Git
- GitHub
- Visual Studio Code

## Cómo ejecutar el programa

1. Clona o descarga este repositorio.
2. Abre la carpeta en Visual Studio Code.
3. Abre una terminal dentro de la carpeta.
4. Ejecuta:

```bash
python main.py
